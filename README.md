# ChatBotOX

# Run
- Clone project.
- Create virtual environment.
- Install requirements:
```
$ pip3 install -r requirements.txt
```


# Data project
#### Run:
```
$ cd data
$ python manage.py runserver
```


#### Admin Area
http://127.0.0.1:8000/

Username: admin
Password: admin

Now, you can generate new messages.

#### Get message
http://127.0.0.1:8000/message/
