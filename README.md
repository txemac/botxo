# ChatBotOX

### Create env and install requirements.
- Clone project.
- Create virtual environment.
- Install requirements:
```
$ pip3 install -r requirements.txt
```


# Data
#### Run:
```
$ cd data
$ python manage.py runserver
```


#### Admin Area
http://127.0.0.1:8000/

Username: admin
Password: admin

Now, you can generate new messages.

#### Get message
http://127.0.0.1:8000/message/


# ChatBot
#### Run:
```
$ cd chatbot
$ python main.py
```


#### Example:
```
$ python main.py
```
